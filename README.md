# Мини-оболочка с файловыми командами

Интерактивный терминал с основными командами Shell/Ubuntu.
В этой оболочке реализованы команды: cat, cd, cp, grep, history, ls, mkdir, mv, rm, touch, undo, zip, tar, unzip, untar.

## Зависимости

Python 3.11+

Для установки остальных зависимостей пропишите следующую команду:
```bash
make setup
```

## Руководство по запуску

Для успешного запуска программы следует прописать в командной строке команду:
```bash
make run
```
