# Мини-оболочка с файловыми командами

Интерактивный терминал с основными командами Shell/Ubuntu.
В этой оболочке реализованы команды: cat, cd, cp, grep, history, ls, mkdir, mv, rm, touch, undo, zip, tar, unzip, untar.

## Зависимости

Python 3.11+

Для установки остальных зависимостей пропишите следующую команду:
```bash
make setup
```

## Руководство по запуску

Для успешного запуска программы следует прописать в командной строке команду:
```bash
make run
```

## Список команд и их функционал

| Команда | Описание | Синтаксис | Опции |
|---------|----------|-----------|-------|
| **cat** | Вывод содержимого файла в консоль | `cat <file1> [file2 ...]` | — |
| **cd** | Переход в указанную директорию | `cd <directory>` | — |
| **cp** | Копирование файла или директории | `cp <source> <destination>` | `-r` — рекурсивное копирование директории |
| **ls** | Вывод списка файлов в директории | `ls [directory]` | `-l` — подробный вывод с правами и размерами |
| **mkdir** | Создание новой директории | `mkdir <directory>` | — |
| **mv** | Перемещение или переименование файла/директории | `mv <source> <destination>` | — |
| **rm** | Удаление файла или директории | `rm <file>` | `-r` — рекурсивное удаление директории |
| **touch** | Создание пустого файла или обновление времени модификации | `touch <file>` | — |
| **grep** | Поиск текста по регулярному выражению в файлах | `grep <pattern> [file ...]` | `-r` — рекурсивный поиск<br>`-i` — игнорировать регистр |
| **zip** | Создание ZIP-архива из директории | `zip <directory> [archive.zip]` | — |
| **unzip** | Распаковка ZIP-архива | `unzip <archive.zip>` | — |
| **tar** | Создание TAR.GZ-архива из директории | `tar <directory> [archive.tar.gz]` | — |
| **untar** | Распаковка TAR.GZ-архива | `untar <archive.tar.gz>` | — |
| **history** | Показать историю выполненных команд | `history [count]` | `count` — количество последних команд (по умолчанию 10) |
| **undo** | Отменить последнюю операцию (cp, mv, rm) | `undo` | — |
| **stop** | Завершение работы программы | `stop` | — |
